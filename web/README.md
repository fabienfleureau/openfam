# Open-F.A.M. Dashboard

The web dashboard for Open-F.A.M. parental control system.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.

## Themes

The dashboard includes three switchable themes:

- **Signal Atlas**: Crisp, technical aesthetic with serif headings
- **Playground Control**: Friendly, professional with rounded elements
- **Nightshift Utilities**: Industrial, rugged with bold dividers

## Build

```bash
npm run build
npm run start
```

## Linting

```bash
npm run lint
npm run type-check
```
